## String Problems

Fill in the code so that it's correct. When correct all will return true.

```ruby
"Hello"[] == "e"
"#Yes We Code"[] == "#"
"#Yes We Code"[] == "d" # Use negative indexing
"Programmer"[4] == ""
"#Yes We Code"[-3] == ""  

"Hello". == "hello"
"world". == "WORLD"
"WeIrD cAsE". == "WEIRD CASE"
"WeIrD cAsE". == "weird case"  

"Hello".index() == 4
"#Yes We Code".index() == 8
"#Yes We Code".index() == 3
"Programmer".index() == 6  

"Hello World".slice() == 'o'
"Hello World".slice() == 'World'
"Dogs are cool"[...9] == 'are'
"Skateboards"[0..] == 'Skate'  

"Hello" +  == "Helloworld"
"#Yes We" +  == "#Yes We Code"
"#Yes ".concat("We Code") == ""
"Pro".concat().concat("mmer") == "Programmer"  
```
